# Project content

## Authentification 
*Persistent User Login Authentication with Redux, JWT Tokens And AsyncStorage.*
- Login
- Register

## Features 
- View all users
- Check messages
- View images in messages
- Send messages
- Send image in message
- Send location link in message 
- View users stories
- Create story
- Edit username
- Edit profile picture
